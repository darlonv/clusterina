# Solicitar acesso ao cluster

Usuários do cluster precisam ser previamente cadastrados, e vinculados a um projeto de pesquisa.

Para cadastrar um projeto de pesquisa,  envie e-mail para:

`email-aqui@ufpr.edu.br`

Informando:

Modelo de e-mail:
```
Título: Solicitação de uso do cluster da UFPR Campus Palotina
Texto:
Solicitação de criação de login para uso do cluster da UFPR Campus Palotina
Nome do projeto: <Nome do projeto>
Descrição do projeto: <Descrição breve do projeto>
Membros do projeto:
- <nome> <e-mail> (<instituição>)
- <nome> <e-mail> (<instituição>)

Coordenador: <nome> <e-mail> (<instituição>)
```

Assim que projeto for cadastrado, os membros do projeto receberão um e-mail com as instruções para acesso.

## Acesso

As instruções de acesso ao cluster são enviadas via e-mail, após o cadastro
