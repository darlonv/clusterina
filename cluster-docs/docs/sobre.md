# Sobre

O cluster HPC é composto por múltiplos nós interconectados, permitindo execução distribuída de tarefas.

## Arquitetura

O cluster é dividido em:

- **Nó mestre**
  - responsável pelo gerenciamento e submissão de jobs
- **Nós de computação**
  - executam as tarefas submetidas
- **Armazenamento compartilhado**
  - diretórios acessíveis pelos nós autorizados

## Tecnologias utilizadas

- Linux
- Slurm
- OpenMPI
- SSH

## Finalidade

O cluster é destinado a atividades de:

- pesquisa
- ensino
- processamento de cargas computacionais intensivas
